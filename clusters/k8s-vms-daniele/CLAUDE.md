# k8s-vms-daniele — App-Specific Gotchas

Terse pointers to release.yml decisions that used to be inline comments — full rationale and any incident detail now lives in Confluence.

- `awx` — the ingress host is Cloudflare-Tunnel-fronted, not exposed via this Ingress's own LB status; requires explicit `external-dns.alpha.kubernetes.io/target` annotation (tunnel target from cluster-vars) to opt in correctly. See [Confluence: AWX](https://fastnetserv.atlassian.net/wiki/spaces/IT/pages/748584961).
- `cloudnative-pg-operator` — this is a different operator from kubenuc's Zalando postgres-operator convention (different CRDs entirely); `podMonitorEnabled` is left off since Prometheus Operator CRDs aren't confirmed present on this cluster. See [Confluence: CloudNativePG Operator](https://fastnetserv.atlassian.net/wiki/spaces/IT/pages/777551916).
- `external-dns` — CF_API_TOKEN needs Reloader wiring to restart on secret rotation; `txtOwnerId` is permanent, changing it orphans every owned record; adoption is opt-in via annotation only. See [Confluence: External DNS](https://fastnetserv.atlassian.net/wiki/spaces/IT/pages/774012929).
- `grafana-alloy` — required secret-key contract on `grafana-cloud-credentials`; collector capabilities must stay at chart defaults or the logs collector breaks; log-severity drop/relabel pipeline keeps active-series budget in check. See [Confluence: Grafana Alloy](https://fastnetserv.atlassian.net/wiki/spaces/IT/pages/748257281).
- `reloader` — needs the dedicated `readOnlyRootFileSystem` top-level flag (not `containerSecurityContext`) or the required `/tmp` mount is never added; UID pinned to house style (65532) instead of the chart's own default. See [Confluence: Reloader](https://fastnetserv.atlassian.net/wiki/spaces/IT/pages/777551893).
- `semaphore` — `admin.*` (not `auth.admin.*`, this chart has no `auth` key); a later 1Password `admin-password` change does NOT propagate to an existing user — use `semaphore user change-by-login` inside the pod instead; `securityContext`/`podSecurityContext` render at the opposite levels from the usual chart convention. See [Confluence: Semaphore](https://fastnetserv.atlassian.net/wiki/spaces/IT/pages/748552193).
